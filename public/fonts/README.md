# SF Pro Display Fonts

Place SF Pro Display and SF Mono font files in this directory.

## Required Files

### SF Pro Display
- SF-Pro-Display-Regular.woff2
- SF-Pro-Display-Medium.woff2
- SF-Pro-Display-Semibold.woff2
- SF-Pro-Display-Bold.woff2

### SF Mono (for code/data)
- SF-Mono-Regular.woff2
- SF-Mono-Medium.woff2
- SF-Mono-Semibold.woff2

## Download

You can download SF Pro fonts from:
- Apple Developer: https://developer.apple.com/fonts/
- SF Pro Display is free for use in apps and websites

## License

SF Pro fonts are licensed by Apple Inc. Please review the license terms before use.

## Fallback

If fonts are not available, the app will fallback to system fonts:
- Sans: -apple-system, BlinkMacSystemFont, system-ui, Segoe UI, Roboto
- Mono: ui-monospace, SFMono-Regular, Menlo, Monaco
